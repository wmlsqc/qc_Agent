# 上下文模块说明

`context/` 目录用于构建传给 Agent 的轻量上下文。它不执行工具，也不替代 ReactAgent，只负责把当前会话中必要的信息整理成简洁、可控的输入。

## 主要文件

- `context_builder.py`：上下文构建函数集合。

## 当前职责

- 裁剪最近会话历史，默认最多保留 10 条消息，约等于最近 5 轮对话。
- 将 Streamlit 中的 `ai` 角色转换为模型侧可识别的 `assistant` 角色。
- 构建当前默认 `user_id` 上下文。
- 调用 `planning/simple_planner.py` 构建轻量规划上下文。
- 调用 `memory/local_memory.py` 构建轻量记忆上下文。
- 生成安全的上下文调试摘要。
- 当记忆读取失败时返回兜底提示，不影响正常聊天。

## 调试摘要

上下文调试摘要包含：

- 当前用户 ID
- 裁剪后的历史消息数量
- 历史消息上限
- 是否存在轻量记忆
- 记忆字段数量
- 当前规划任务类型

调试摘要不会包含：

- 完整原始聊天历史
- 记忆字段的具体值
- API Key
- 用户自由输入原文
- 其他敏感信息

## 使用位置

`services/chat_service.py` 会调用本模块，将会话历史、用户 ID、轻量记忆和轻量规划结果传给 ReactAgent。
